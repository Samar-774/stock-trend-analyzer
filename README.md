# Stock Market Trend Analyzer

Live stock data, technical indicators, and 30-day forecasting — built with Streamlit, yfinance, and Prophet.

## Status
🚧 In progress — Hour 1/5 (project setup + data fetching) complete.

## Setup
```bash
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```
